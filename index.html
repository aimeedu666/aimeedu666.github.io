- PROJECT SUMMERY -

Topic

My website allows users to search for detailed information on food products listed in the United States, including product name, category, labels, and nutrition facts. The home page will automatically displays 6 most popular products currently. Users can also add products to their favorites list, and create, edit, delete items within that list.

Food health issues are gaining more and more attention. My website hopes to give people a tool to get information about food.

Design & Database

My website uses a food search API to get search results. It uses front-end API as well as back-end API. All the data for the favorite list is stored in a database using SQL. However, it takes some time to get the results because I uses the free version of the API. That's why I added a javascript plugin when a page is loading.

The website design used bootstrap framework combined with css. The website is responsively designed, which means it's adapted to three diffenrent screen sizes of mobile, tablet, and desktop.

A reminder: refreshing every page after making any change to the "favorite list".
